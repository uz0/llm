# Privacy Policy for LLM Agents Workflow Course

**Last updated:** December 2025
**Effective date:** December 2025

## Overview

This privacy policy describes how the LLM Agents Workflow Course website ("the Site") collects, uses, and protects information when you visit our website hosted at https://llm.uz0.dev.

## Data Collection

### Analytics Data
We use Google Tag Manager (GTM-PMQLRD6K) to collect anonymized website usage data for the purpose of improving our educational content and user experience.

**What we collect:**
- Page visits and navigation patterns
- Session duration and engagement metrics
- General geographic location (country/region level)
- Browser type and device information
- Referral sources

**What we don't collect:**
- Personally identifiable information (PII)
- IP addresses (anonymized at collection)
- User names or email addresses
- Course content or personal data
- Detailed location information

### No User Accounts Required
This course website does not require user registration, login, or account creation. All content is freely accessible without providing personal information.

## Cookies

The Site uses cookies and similar tracking technologies to:
- Analyze website traffic and usage patterns
- Improve site performance and user experience
- Understand which content is most valuable to learners

**Cookie types:**
- **Analytics cookies**: Essential anonymous usage tracking
- **Functional cookies**: Maintain user preferences (theme selection)
- **Security cookies**: Protect against malicious activities

### Cookie Consent
By continuing to use this website, you consent to the use of analytics cookies as described in this policy. You can disable cookies through your browser settings, though this may affect website functionality.

## Data Security

We implement appropriate security measures to protect collected data:
- **HTTPS encryption** for all data transmission
- **Anonymization** of user identifiers
- **Limited data retention** - only keeping data necessary for analytics
- **No third-party data sharing** beyond analytics processing

## Your Rights

Under GDPR and other privacy regulations, you have the right to:
- **Access**: Request information about data we collect
- **Rectification**: Correct inaccurate data
- **Erasure**: Request removal of your data (though our data is anonymized)
- **Restriction**: Limit processing of your data
- **Portability**: Request data transfer (not applicable as we don't collect personal data)
- **Objection**: Object to data processing

## International Data Transfers

Data may be processed and stored in servers located outside your country. We ensure appropriate safeguards are in place for international data transfers in compliance with applicable privacy laws.

## Children's Privacy

This website is intended for adult learners and professionals. We do not knowingly collect information from children under 13 years of age. If you believe we have collected information from a child, please contact us.
